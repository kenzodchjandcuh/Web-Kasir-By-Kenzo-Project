Panduan Koneksi Cloud

Bagaimana cara menyambungkan Firebase?

  1. Kunjungi Firebase Console.
  2. Buat proyek baru, lalu tambahkan aplikasi web ke dalam proyek tersebut.
  3. Salin potongan objek konfigurasi Web SDK. Formatnya seperti ini: 
    {
  "apiKey": "AIzaSy...",
  "authDomain": "app.firebaseapp.com",
  "projectId": "app-project-id",
  ...
}
  4. Pergi ke menu Pengaturan di AeroPOS, tempel kode JSON di atas dan klik Sambungkan Firebase.
  5. PENTING (Aturan Realtime Database): Di Firebase Console Anda, pilih Realtime Database > tab Rules (Aturan), ubah konfigurasinya menjadi publik agar sinkronisasi dari kasir diizinkan: 
    {
  "rules": {
    ".read": "now < 1781974800000", // Sampai 2026-06-21
    ".write": "now < 1781974800000" // Sampai 2026-06-21
  }
}
    Lalu klik Publish.
  6. Unggah data demo awal ke awan Firebase dengan mengeklik tombol Migrasi Data.

*Data Anda akan tetap tersimpan di browser secara offline walaupun koneksi internet terputus, dan otomatis disinkronkan kembali saat online.
